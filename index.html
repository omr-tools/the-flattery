Supports: Blender UV seams · named "seam" groups · boundary edges
i How seam detection works The app detects UV seams — edges where two adjacent faces have different UV coordinates for the same 3D vertices.

This is how Blender marks seams when you use Edge → Mark Seam and unwrap with UV → Unwrap. The resulting UV layout is used directly as the flat 2D pattern.

For OBJ files without UV coords, named seam groups or boundary edges are used instead.
